# Translations

This folder contains Drupal interface translation `.po` files sourced from localize.drupal.org, with usage notes summarized from https://drupalbook.org/localize-drupal.

## How to use
1. Enable the **Interface Translation** module (it will enable **Language** automatically).
2. Add your language at `/admin/config/regional/language` (Configuration > Region and language > Languages).
3. Import the matching `.po` file at `/admin/config/regional/translate/import` (Configuration > Region and language > User interface translation > Import).
4. Drupal will update translation strings in a batch process.
